## ✨ Features

| | |
|---|---|
| 🏝️ **Dynamic Island** | Floating animated overlay that lives at the top of your screen |
| 🎨 **Smooth Animations** | Physics-based spring animations and fluid transitions |
| 🧩 **Modular Widgets** | Add, remove and rearrange widgets — time, weather, media, CPU/RAM, battery and more |
| 🌍 **Multi-language** | English, Turkish, French, Russian, Spanish, German, Japanese, Chinese, Arabic, Azerbaijani |
| 🎨 **Theming** | Light, dark, and fully custom theme support |
| ⏱️ **Timer · Stopwatch · Alarm** | Built-in multi-mode timer widget |
| 🔇 **Low footprint** | Lightweight — stays out of your way until you need it |

---

## 🚀 Getting Started

### Requirements
- Windows 10 or 11 **(64-bit)**

### Installation

1. Visit the [**Microsoft Store page**](https://apps.microsoft.com/detail/9P0JFN5L96VH?hl=tr-tr&gl=TR&ocid=pdpshare) and click **Get**
2. Daylight will appear at the **top center** of your screen — that's it

---

## 🖱️ How to Use

| Action | Result |
|---|---|
| **Hover** over the island | Expands to show widgets |
| **Double-click** the island | Opens the home menu |
| **Right-click** system tray icon | Settings, quit, and more |
