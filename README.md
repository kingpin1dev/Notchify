# Daylight

A dynamic island-style desktop overlay for Windows — bringing a macOS Dynamic Island experience to your PC.

## Features

- **Dynamic Island** — A floating, animated overlay that sits at the top of your screen
- **Widgets** — Time, weather, media playback, CPU/RAM usage, battery, shortcuts, and more
- **Smooth Animations** — Physics-based spring animations and fluid transitions
- **Theming** — Light, dark, and custom theme support
- **Multi-language** — English, Turkish, French, Russian, Spanish, German, Japanese, Chinese, Arabic, Azerbaijani
- **Modular** — Add, remove, and rearrange widgets freely
- **Timer / Stopwatch / Alarm** — Built-in multi-mode timer widget
- **Low footprint** — Lightweight and stays out of your way

## Requirements

- Windows 10 / 11 (64-bit)
- [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

## Installation

1. Download the latest `Daylight_vX.X_Setup.exe` from [Releases](../../releases)
2. Run the installer
3. Daylight will appear at the top center of your screen

## Usage

- **Hover** over the island to expand it
- **Double-click** to open the home menu
- **Right-click** the system tray icon to access settings or quit
